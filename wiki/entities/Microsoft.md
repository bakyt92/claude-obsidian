# Microsoft

**What**: Global technology company. In this vault it appears through three distinct arms, so this page is the shared home for all of them (aliases route `Microsoft Research` and `Azure Cloud Advocates` here).

## Arms relevant to this vault

- **Azure Cloud Advocates** — Microsoft's developer-relations group, maintainers of the [[Data Science for Beginners]] curriculum and the broader **"For Beginners"** open-education series (ML, AI, Web Dev, IoT, Cybersecurity, Generative AI, LangChain for Beginners). All MIT-licensed, project-based, multilingual.
- **Microsoft Research (Cambridge, UK)** — the lab where [[Christopher Bishop]] was Assistant Director when he wrote [[Pattern Recognition and Machine Learning]]. (PRML's `[[Microsoft Research]]` link resolves here via alias.)
- **Azure / Azure Machine Learning** — the cloud platform used in the *For Beginners* cloud lessons (Azure ML Studio, low-code training and deployment).

## Why it matters here

- Provider of a major **open-education** footprint (the *For Beginners* curricula) — a real-world parallel to this vault's own compounding-knowledge ethos.
- Publisher/host context for two vault sources: [[Data Science for Beginners]] (Azure Cloud Advocates) and [[Pattern Recognition and Machine Learning]] (Microsoft Research, via [[Springer]]).
