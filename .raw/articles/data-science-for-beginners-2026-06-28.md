# Data Science for Beginners — A Curriculum

**Maintainer**: Microsoft (Azure Cloud Advocates)
**Format**: open GitHub curriculum · 20 lessons over 10 weeks · MIT License · ~35.8k stars
**Tech**: Python (Pandas, Matplotlib), SQL + NoSQL, Jupyter notebooks, Azure ML Studio, GitHub Codespaces
**Pedagogy**: project-based; each lesson has pre/post quizzes, written instructions, knowledge checks, challenges, assignments; sketchnotes + optional videos; 50+ language translations via GitHub Actions.

## Lesson list (6 sections, 20 lessons)

### Introduction (1–4)
1. Defining Data Science (core concepts; AI, ML, big data)
2. Data Science Ethics (ethics frameworks and challenges)
3. Defining Data (classification and sources)
4. Statistics & Probability (mathematical foundations)

### Working With Data (5–8)
5. Relational Data (SQL)
6. NoSQL Data (document/non-relational databases)
7. Working with Python (Pandas, data exploration)
8. Data Preparation (cleaning, transforming)

### Data Visualization (9–13)
9. Visualizing Quantities (Matplotlib)
10. Visualizing Distributions
11. Visualizing Proportions
12. Visualizing Relationships (correlations)
13. Meaningful Visualizations

### Data Science Lifecycle (14–16)
14. Introduction (acquisition / extraction)
15. Analyzing
16. Communication (presenting insights)

### Data Science in the Cloud (17–19)
17. Cloud Data Science Intro (benefits of cloud)
18. Low-Code Training (low-code model training)
19. Azure Deployment (Azure ML Studio)

### Data Science in the Wild (20)
20. Real-World Examples (applied projects)
