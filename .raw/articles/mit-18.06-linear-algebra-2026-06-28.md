# MIT 18.06 Linear Algebra

Course home page for MIT's undergraduate Linear Algebra course, taught by **Gilbert (Gil) Strang**. The page serves as the hub linking the current-semester GitHub repository, archived problem sets and exams, and the famous video-lecture series.

## Instructor
- Gilbert Strang (Professor of Mathematics, MIT)

## Course materials linked from the page
- **Current semester**: GitHub repository `mitmath/1806` (psets, exams, Julia computational notebooks)
- **OpenCourseWare**: Spring 2010 archived materials with full lecture videos
- **Video lectures**: 34 YouTube-hosted lectures (the canonical Spring 2005 recordings)
- **Archived problem sets and exams** with solutions
- **Julia** computational examples linked via GitHub

## Syllabus (34 lectures)
- Geometry of linear equations; elimination with matrices; matrix multiplication
- LU factorization (A = LU); transposes; permutations
- Vector spaces and subspaces; column space and nullspace
- Solving Ax = 0 (pivot/free variables) and Ax = b (complete solution)
- Independence, basis, and dimension
- The four fundamental subspaces
- Matrix spaces; rank-one matrices; small-world graphs / networks
- Orthogonality of the four subspaces
- Projections onto subspaces; least squares and projection matrices
- Orthogonal matrices and Gram–Schmidt (A = QR)
- Determinants: properties, formulas, cofactors; Cramer's rule, inverse, volume
- Eigenvalues and eigenvectors; diagonalization (A = SΛS⁻¹); powers Aⁿ
- Differential equations and exp(At); Markov matrices; Fourier series
- Symmetric matrices and positive definiteness
- Complex matrices; the Fast Fourier Transform (FFT)
- Positive definite matrices and minima
- Similar matrices and Jordan form
- Singular Value Decomposition (SVD)
- Linear transformations and their matrices; change of basis
- Left/right/pseudo inverses

## Exam structure
- Three quizzes (Quiz 1, Quiz 2, Quiz 3), each with dedicated review lectures, plus a final exam.
